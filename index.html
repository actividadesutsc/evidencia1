Mascotas
Perro Gato Cocodrilo Leon

Objetivo: Desarrolla interfaces de usuario para aplicaciones móviles utilizando: - HTML5. - CSS3. - Java Script Actividad por desarrollar Elaborará una página en HTML puro aplicable a cualquier dispositivo Instrucciones: 1. Elaborar página en la que se presten diferentes tipos de mascotas (Perro, Gato, Aves, Reptiles, etc.), que esté en HTML puro. 2. Dar formato a la página de la actividad anterior, para que se pueda mostrar tanto en una computadora, como una Tablet y un smartphone. 3. Habilitar efectos dinámicos a la página utilizando JavaScript, por ejemplo: que, al seleccionar opciones de menú, se cambien los colores y estilos de la letra del menú, etc. 4. Comparte la liga y adjunta pantallazos como evidencia.
